Control Key relocation

Tab acts as Control key when held

Allows you to retain the tab key, but have a control key when held. Useful if the capslock key is already in use as the fn key with the other mods here.

No stretch mods

Uses fn as a modifier

Includes modifications that prevent the usual stretches for the following keys:

  • backspace
  • escape
  • arrow keys (uses vim mappings)
  • grave (accent)

Works well when fn key is remapped to the caps_lock key, since function is usually a stretch by default.

Move fn key

Move fn to replace caps_lock key

A common mod is to move the control key to where the caps lock key is, but moving the function key instead allows us to remap other keys without altering shortcuts that use the control key.
